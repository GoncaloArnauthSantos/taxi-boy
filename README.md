## **Getting Started**

1. Clone the repository:
   ```bash
   git clone https://github.com/GoncaloArnauthSantos/taxi-boy.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Visit the app at `http://localhost:3000`.
